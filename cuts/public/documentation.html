myFlix

This app will provide you information about different movies, directors, and genres. You will be able to sign up, update your personal information, and create a list of your favorite movies.

Description Endpoint URL Type Request body Response body
Get a list of all movies /movies GET None JSON object holding data about all the movies, containing the title, genre and director's name property. Example:
{
"Title": "The Terminator",
"Description"....":
"Genre": "Sci-Fi",
Name :"James Cameron"
},
Get data about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing a title, description, genre property, director property and image. Example: { "Genre": { "Name": "Thriller", "Description": "Thriller film, also known as suspense film or suspense thriller, is a broad film genre that involvesexcitement and suspense in the audience." }, "Director": { "Name": "Jonathan Demme", "Bio": "Robert Jonathan Demme was an American director, producer, and screenwriter.", "Birth": "1944", "Death": "2017" }, "Actors": [], "_id": "5d9bab4307ab30237c44c353", "Title": "Silence of the Lambs", "Description": "A young FBI cadet must receive the help of an incarcerated and manipulative cannibal killer to help catch another serial killer.", "ImagePath": "silenceofthelambs.png", "Featured": true }

Get data about a movie genre by genre name /movies/genres/[Name] GET None JSON object holding data about a movie genre. Example: { "Name": "Action", "Description": "Action film is a film genre in which the protagonist or protagonists are thrust into a series of events that typically include violence, extended fighting, physical feats, and frantic chases." }
Get data about a director by name /movies/directors/[Name] GET None JSON object holding data about a director. Example: {"Name": "Jon Favreau", "Bio": "Jonathan Favreau is an American actor, director, producer, screenwriter, and chef.", "Birth": "1966" }
Add a new user /users POST JSON object holding data about the user to add. Example:
{
"Username": "arunkumar_rl",
"Password": "arunkumarrl",
"Email": "arunkumar_rl@yahoo.com",
"Birthday": "1988-06-16"
} 		JSON object holding data about the user that was add, including an ID. Example:

Update a user information /users/[Username] PUT JSON object holding data about the user to add. Example:
{
"Username": "arunkumar_rl"(required),
"Password": "arunkumarrl"(required),
"Email": "arunkumar_rl.yahoo.com"(required),
"Birthday": "1988-06-16"

Add a movie to the list of favorites /users/[Username]/Movies/[MovieID] POST JSON object holding data about a movie which users like to add to their list of favorites. JSON object holding data about the movie that was added to the list of favorite.
Delete a movie from the list of favorites /users/[Username]/Movies/[MovieID] DELETE None A text message indicating the movie that was removed from the list of favorites.
Remove a user /users/[Username] DELETE None A text message indicating the user who was removed
